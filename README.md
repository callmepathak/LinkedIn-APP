# LinkedIn Clone - Backend

A microservices-based LinkedIn clone built with **Java 21**, **Spring Boot 3**, and **Spring Cloud** (Eureka, Gateway, OpenFeign). Services use **PostgreSQL** (JPA) and **Neo4j** for data, **Apache Kafka** for async messaging, and **JWT** + **jBCrypt** for authentication. File uploads are handled via **Cloudinary** and **Google Cloud Storage**. API documentation is aggregated via **SpringDoc OpenAPI** at the Gateway.
